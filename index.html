Car Rental Management System

Streamlining Operations for a Digital Garage

The Problem

Small to medium-sized car rental companies often rely on manual record-keeping with spreadsheets or paper-based systems. This leads to inefficiencies, difficulty tracking vehicle availability, higher risk of human error in billing, and a poor overview of business performance.

The Solution

I designed and developed a comprehensive desktop application to serve as a centralized management hub. The system digitizes the entire rental lifecycle, from adding new vehicles to generating end-of-day financial reports, ensuring data accuracy and saving administrative time.

Key Features & Implementation

  • Vehicle & Customer Management: Secure databases to add, edit, and archive vehicle and customer records.
  • Transaction Processing: An intuitive interface to handle rentals and returns, automatically calculating costs.
  • Availability Tracking: Automatic status updates (Available, Rented, Maintenance) to prevent double-booking.
  • Reporting Module: Generate instant reports on revenue and customer history.
  • Tech Stack: Built with Java and Java Swing for the GUI, connected to a MySQL database.

What I Learned

This project deepened my understanding of relational database design, CRUD operations, and building intuitive user interfaces for complex business logic.

Pizza Ordering System

The Problem

Traditional phone orders for pizza can be prone to errors during peak hours, lack a visual menu, and don't provide an easy way for customers to customize their orders or for owners to track sales trends.

The Solution

I created a user-friendly desktop application that simulates a modern pizza ordering experience. It allows customers to build their perfect pizza with a visual list of toppings, manages order fulfillment for the kitchen staff, and provides administrators with tools to update the menu and view sales data.

Key Features & Implementation

  • Interactive Menu: Graphical interface to select sizes, crust types, and toppings with real-time cost calculation.
  • Order Management Cart: Users can add multiple pizzas, review, and proceed to checkout.
  • Admin Dashboard: Secured portal for managers to update the menu and view order analytics.
  • Order Tracking: Simple status updates (e.g., "Order Received," "Cooking," "Ready for Pickup").
  • Tech Stack: Developed with Java and Java Swing. Utilized MySQL to manage items and orders.

What I Learned

This project was a fantastic lesson in event-driven programming and creating a dynamic user experience (UX). Handling real-time price calculation reinforced my logic skills in Java.

Pine City Zoo

Bringing the Zoo Experience Online

The Problem

Potential visitors often want to preview a zoo's attractions before planning a visit. A static, informational website fails to capture the excitement and educational value of the animals, missing an opportunity to engage users and drive ticket sales.

The Solution

I designed and developed a dynamic, interactive virtual tour website for a fictional zoo. The site allows users to explore different animal exhibits, learn detailed facts in an engaging way, and easily find practical visit information, creating an online experience that mirrors the wonder of an actual visit.

Key Features & Implementation

  • Interactive Habitat Sections: Themed sections (e.g., African Savannah, Amazon Rainforest) with curated animal displays.
  • Dynamic Animal Cards: Clickable animal profiles that reveal detailed information about their species, habitat, and conservation status, enhancing user engagement.
  • Responsive Design: A fully responsive layout that provides an optimal viewing experience across all devices, from desktops to mobile phones.
  • Modern Front-End Stack: Built from scratch with semantic HTML5, styled with CSS3 (utilizing Flexbox/Grid for layout), and enhanced with interactive elements using vanilla JavaScript.
  • Design-First Approach: Prototyped the user interface and experience in Figma before writing any code, ensuring a visually appealing and intuitive final product.

What I Learned

This project was a perfect exercise in modern front-end development. It solidified my skills in creating responsive layouts, manipulating the DOM with JavaScript to create interactivity, and following a design-to-code workflow. It emphasized the importance of designing for the user first to create a compelling and functional web experience.
